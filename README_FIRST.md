# 使用方式

1. 新建一个空目录，并进入目录。
2. 将本压缩包中的 `AGENTS.md`、`docs/` 和 `CODEX_START_PROMPT.md` 放到仓库根目录。
3. 在 Codex 中打开该目录。
4. 使用 Plan mode。
5. 把 `CODEX_START_PROMPT.md` 的内容发送给 Codex。
6. Phase 0 完成后，检查说明是否准确；确认后输入：`继续 Phase 1`。
7. 每个 Phase 均按同样方式推进，不允许 Codex 一次性做完整项目。

推荐在每个 Phase 结束时自己完成三件事：

- 运行 App，手动验证。
- 用自己的话复述本阶段状态流和设计理由。
- 保存一条简短学习笔记，记录“我原来不知道什么、现在知道什么、还不清楚什么”。
